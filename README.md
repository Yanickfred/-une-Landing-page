# Page d'atterrissage (Landing Page)

## Présentation

Petite page web statique destinée à présenter un produit, un service ou un portfolio.

## Technologies

- HTML
- CSS
- Ressources statiques dans le dossier `assets/`

## Installation et aperçu local

### Prérequis

- Un navigateur web moderne (Chrome, Firefox, Edge).

### Ouvrir localement

- Ouvrir `page.html` directement dans le navigateur.

### Lancer un serveur local (optionnel)

Avec Python 3 :

```bash
python3 -m http.server 8000
# puis ouvrir http://localhost:8000/page.html
```

Avec l'extension Live Server (VS Code) : clic droit sur `page.html` → "Open with Live Server".

## Structure du projet

- `page.html` : page principale
- `styliseur.css` : feuilles de style
- `assets/` : images, icônes, polices et autres ressources

## Utilisation

- Modifier le contenu : éditer `page.html`.
- Modifier le style : éditer `styliseur.css`.
- Ajouter des ressources : placer les fichiers dans `assets/` et référencer depuis `page.html` ou `styliseur.css`.
